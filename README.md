# Task 4: Classification with Logistic Regression

## Objective
Build a binary classification model using Logistic Regression.

## Dataset
Breast Cancer Wisconsin Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
- Loaded dataset
- Split into training and testing sets
- Standardized features
- Trained Logistic Regression model
- Evaluated using:
  - Confusion Matrix
  - Precision
  - Recall
  - ROC-AUC
- Plotted ROC Curve
- Explained Sigmoid Function
- Demonstrated threshold tuning

## Results
The Logistic Regression model achieved high classification performance on the Breast Cancer Wisconsin dataset and was evaluated using multiple metrics.
